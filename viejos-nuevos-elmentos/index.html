etiqueta mark (texto marcado/resaltado)

la etiqueta mark fue creada para resaltar texto,
esta se pude emplear con mas frecuencia para resaltar algun tipo
de palabra clave con la que le usuario busque en la pagina, resaltando
los resultados similares que se obtienen, por ejemplo:
palabra clave: carro
resultado: un carro rojo en la autopista

Etiqueta i (texto en cursiva sin importancia)

la etiqueta i al igual que b se utiliza cuando no halla una etiqueta
mas apropiada para el caso, como un strong, em, mark, etc, se utiliza para referirse a un tema tecnico o una frase en otro idioma, un pensamiento, etc

la etiqueta small se utliza en HTML5 para presentar texto mas pequeño
para cuestiones de terminos legales, derechos de autor y demas

Etiqueta address (informacion de contacto del autor)

esta etiqueta se usa para agregar informacion de contacto del autor/propietario
de un documento o articulo, o electrónico, URL, dirección física, número de teléfono,
identificador de redes sociales, etc. esta etiqueta de por lo general
deberia ser incluido en el footer ya sea el de la pagina si la informacion
de contacto es la del dueño de la pagina, o en un footer de un article
si la informacion de contacto es la de la idea que se experesa en el, siendo de otra persona

Etiqueta time

en los ateriores ejmplos de articles se colocaba la fecha de publicacion
con una etiqueta p, pero esto se de realizar con la tiqueta time, que nos permite
declarar un texto compresimple tanto para los navegadores como para el usuario que represente la fecha
el atributo datetime en la etiqueta time representa la fecha comprensible para el navegador, siguendo
un patron similar a 2011-10-12T12:10:45.
el atributo pubdate se utiliza para indicar que el valor del atributo datatime representa la fecha de publicacion
